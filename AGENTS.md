# AGENTS.md

## 🎯 Project Overview
This project is a personal portfolio website for an IoT & AI Engineer with a focus on clean, futuristic dark UI design.

The goal is to showcase engineering projects in a professional and modern way without using complex dashboards.

---

## 🧩 Tech Stack
- HTML / Tailwind CSS (or React + Vite)
- Optional: JavaScript for interactivity
- Deployment: Netlify / Vercel / GitHub Pages

---

## 🎨 Design Principles
- Dark mode (primary background: #0b0f19)
- Minimalist but futuristic
- Subtle neon accents (blue / purple)
- Clean typography
- Avoid clutter and unnecessary elements

---

## 🏗️ Structure
Sections must follow this order:

1. Hero
2. About
3. Projects
4. Tech Stack
5. Contact

---

## ⚙️ Development Rules

### General
- Keep code clean and modular
- Use semantic HTML
- Avoid inline styles (use Tailwind or CSS classes)

### UI/UX
- Prioritize readability
- Maintain consistent spacing
- Ensure responsive design

### Animations
- Use subtle animations only
- Avoid heavy or distracting effects

---

## 📦 Components Guidelines

### Hero
- Clear headline
- Short subtitle
- CTA button

### Projects
- Card-based layout
- Each project must include:
  - Title
  - Description
  - Tech stack
  - External link (GitHub/demo)

### Tech Stack
- Use icons where possible
- Display in grid layout

---

## 🚫 Restrictions
- Do NOT include IoT dashboards or live data visualization
- Do NOT overload with animations
- Do NOT use bright backgrounds

---

## ✅ Quality Checklist
- Responsive on all screen sizes
- Fast loading
- Clean visual hierarchy
- Professional tone
